Soli PHP Framework
--------------------

Soli 是一个轻量级的 PHP 框架，框架的设计，意在松耦合、可扩展、简洁易用。

## 提供的功能

MVC、[依赖注入]、[事件管理]、[闪存消息]、[模版引擎]（[Twig]、[Smarty]）、[命令行应用]等功能

## 使用

详见 [https://github.com/ueaner/soliphp]
